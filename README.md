# Lock In!
A website/game that rewards you for getting your work done.

# Overall structure

- Add tasks
    - text box (make it required)
    - submit button
    - add()
        - extract text box contents
        - task put in a list below the task box
- Choose a task
- Start 15 minute timer if a task is chosen
    - Otherwise show error
- When time is up, add $100 to user’s balance
- Balance is stored in local storage
